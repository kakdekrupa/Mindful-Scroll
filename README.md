# Mindful Scroll

Mindful Scroll is a Chrome browser extension that encourages intentional and mindful browsing. Instead of blocking websites, the extension helps users reflect on their browsing habits by prompting them to set an intention before starting a browsing session and reflect on the outcome afterward.

The goal of this project is to promote awareness and reduce unconscious scrolling by introducing a short check-in process before and after a browsing session.

---

## Features

- **Intentional Browsing Start**  
  Users select their current mood and set an intention before starting a browsing session.

- **Session Timer**  
  The extension tracks the duration of each browsing session in real time.

- **Post-Session Reflection**  
  After ending a session, users record how they feel and whether the session was helpful.

- **Session History**  
  The extension saves recent session records so users can review their browsing patterns.

- **Privacy Focused**  
  All data is stored locally using `chrome.storage.local`. No external servers or tracking are used.
